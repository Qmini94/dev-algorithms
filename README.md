# dev-algorithms

java + JUnit 기반으로 코딩테스트 문제를 TDD로 풀이하고, 
풀이 과정과 개념을 Markdown으로 문서화하는 저장소

---

## Repository Structure

- `src/main/java` : 문제 풀이 코드(솔루션)
- `src/test/java` : JUnit 테스트(TDD)
- `docs/` : 알고리즘 개념 정리 및 공통 템플릿
- `problems/` : 문제별 풀이 문서 (템플릿 기반으로 누적)
